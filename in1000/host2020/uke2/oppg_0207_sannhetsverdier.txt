02.07: Sannhetsverider

Filnavn: sannhetsverdier.py

Oppgaven løses med penn og papir/i et tekstdokument, men sjekk gjerne om du får
til å programmere en løsning for å sjekke svaret til slutt.

Gitt at verdien av b = False, og verdien av x = 0.
Hva er sannhetsverdien til følgende uttrykk?

b
x == 0
b and x == 0
b or x == 0;
not b and x == 0
not b or x == 0
b and x != 0
b or x != 0
not b and x != 0
not b or x != 0
